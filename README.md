# SAAT 学格 — Student Academic Aptitude Type

> 发现你的学格，找到你的方向 · [saattype.com](https://saattype.com)

SAAT 是一个面向准高中生和准大学生的学业倾向测评系统，通过四维度科学模型，帮助学生了解自己的学习风格，获取个性化的分科/专业推荐。

## ✨ 特性

- 🎯 **20道题 · 3分钟** — 快速完成测评
- 🧬 **4维度 · 16种学格** — 科学的学习倾向分类体系
- 📊 **雷达图可视化** — 直观展示你的四维学格画像
- 🎓 **分科/专业推荐** — 基于学格匹配个性化的学业建议
- 📤 **一键分享** — 生成专属学格卡片，和朋友PK
- 🤖 **AI深度报告** — DeepSeek驱动的定制化分析（付费 ¥9.9）

## 🚀 快速开始

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)

## 🧬 四维度体系

| 维度 | 字母 | 中文 |
|------|------|------|
| 学习节奏 | S / D | 稳步型 vs 灵动型 |
| 思维通道 | R / C | 理性型 vs 创造型 |
| 知识偏好 | T / P | 理论型 vs 实践型 |
| 动力来源 | I / E | 内驱型 vs 外驱型 |
